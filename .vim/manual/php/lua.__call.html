Lua::__call

(PECL lua >=0.9.0)

Lua::__call — Magic method for Lua to call Lua functions

Description

public void Lua::__call ( string $lua_func , array $args )

Warning

This function is currently not documented; only its argument list is available.

Parameters

lua_func

Function name in lua

args

Arguments passed to the Lua function

Return Values

Examples

Example #1 Lua::call()example

<?php
$lua = new Lua();
$lua->eval(<<<CODE
 function dummy(foo, bar) 
    print(foo, ",", bar)
 end
CODE
);
$lua->dummy("Lua", "geiliable"); 
?>

The above example will output something similar to:

Lua,geiliable
